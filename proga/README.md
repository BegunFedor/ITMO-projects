Programming Labs

В этой папке собраны лабораторные работы по программированию, выполненные в рамках обучения в Университете ИТМО.

Описание

Лабораторные охватывают основные темы курса программирования на Java: от базовой работы с массивами и вычислениями до объектно-ориентированного проектирования, клиент-серверной архитектуры, работы с базами данных и создания графического интерфейса.

Проекты в этой папке отражают постепенное усложнение задач и развитие практических навыков разработки: построение объектных моделей, работа с коллекциями, сетевое взаимодействие, хранение данных, проектирование архитектуры приложения и развитие пользовательского интерфейса.

Структура
	•	lab1/ — вычисление математических выражений, работа с массивами и форматированным выводом
	•	lab2/ — объектно-ориентированная модель покемонов и реализация атак
	•	lab3-4/ — разработка объектной модели по текстовому описанию предметной области
	•	lab5/ — консольное приложение для управления коллекцией объектов
	•	lab6/ — развитие приложения до клиент-серверной архитектуры
	•	lab7/ — интеграция приложения с PostgreSQL и работа с хранением данных
	•	lab8/ — графический клиент для системы управления коллекцией

Темы лабораторных работ

Lab 1

Реализация программы на Java для вычисления выражений с использованием одномерных и двумерных массивов, математических функций стандартной библиотеки и форматированного вывода результата.

Lab 2

Изучение основ ООП на примере модели покемонов: наследование, полиморфизм, классы атак, работа с внешней библиотекой и симуляция боя.

Lab 3–4

Построение объектной модели по литературному фрагменту. Акцент на проектировании классов и интерфейсов, корректной иерархии, сценарии взаимодействия объектов, исключениях, enum, record и применении принципов ООП.

Lab 5

Разработка консольного приложения для управления коллекцией объектов. Использование TreeSet, командного интерфейса, чтения и записи данных в файл, валидации ввода и обработки ошибок.

Lab 6

Переход от локального приложения к клиент-серверному взаимодействию. Реализация обмена данными по сети, сериализации объектов, разделения на клиентскую и серверную части.

Lab 7

Расширение клиент-серверного приложения за счёт подключения базы данных PostgreSQL. Реализация слоя доступа к данным, хранения пользователей и объектов, а также обработки авторизации.

Lab 8

Разработка графического интерфейса клиента. Реализация таблиц, визуализации объектов, авторизации, редактирования данных и взаимодействия с серверной частью приложения.

Назначение папки

Папка proga используется как единое место для хранения лабораторных работ по программированию, чтобы сохранять структуру учебных проектов, отслеживать развитие навыков и иметь удобную точку доступа ко всем основным заданиям курса.
