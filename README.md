ITMO Projects

Репозиторий с учебными проектами, выполненными в рамках обучения в Университете ИТМО.

О репозитории

Здесь собраны проекты по программированию, алгоритмам, базам данных и веб-разработке.
Основной акцент — на Java, объектно-ориентированном программировании, клиент-серверной архитектуре, работе с базами данных и прикладных учебных задачах.

Структура репозитория
	•	proga/ — лабораторные работы по программированию на Java
	•	algorithms/ — задачи и проекты по алгоритмам и структурам данных
	•	databases/ — работы, связанные с SQL, проектированием БД и интеграцией с СУБД
	•	web/ — учебные веб-проекты и эксперименты с клиент-серверной логикой

Содержимое папки proga

В папке proga собраны лабораторные работы по курсу программирования в ИТМО.

Примеры тем:
	•	работа с массивами и математическими выражениями
	•	объектно-ориентированное программирование
	•	моделирование предметной области
	•	управление коллекциями
	•	клиент-серверное взаимодействие
	•	интеграция с PostgreSQL
	•	графический интерфейс пользователя

Для каждой лабораторной работы добавлен отдельный каталог с исходным кодом и кратким описанием.

Цель репозитория

Этот репозиторий используется как единая точка хранения учебных проектов, чтобы:
	•	поддерживать порядок в работах по разным дисциплинам
